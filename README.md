# What is Base-Negative Complex?
Base-Negative Complex (BNC) is a new math system developed by howsworkzco.  
This math system allows us to visuaalize non-existent numbers that  
wouldn't be in our normal mathematics.  

**For Example:**  
• Five hundred and negative one.    
• Six hundred and negative twenty.  

These examples are not equations, but rather actual  
proper numbers in the BNC system.  
(Five hundred and neg. one ≠ 500 + -1)
## The Writing System
The writing system for BNC is easy.   
**Here's an example:**
five hundred and negative one: 50<-1>  
You use angled brackets ("<>") to represent it.  
***
# Base < offset >  
***
## Base
Base is the normal number.  
## Offset
Offset is the "negative one" in the "500". 
# Offset rules
Each offset ranges from 9 to -9. (Ranges from 0 to 9 in BNC-M)
